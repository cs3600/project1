Shell Project

Features:

launch basic commands with optional flags, such as:

  ls
  
  ls -l
  
  pwd

handle redirections:

  ls > out.txt
  
  ls 1> out.txt
  
  ls 2> error.txt
  
  kill -9 < procId.txt
  
handle piped commands:

  ls | wc
  

Additional features that may be supported:

  More Error Handling.
  
  cd command.
